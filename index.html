Web Scraping Cambridge School Data in Indonesia

Developed an automated web scraping script using Playwright to extract Cambridge school data in Indonesia. The script dynamically selects Indonesia as a region, retrieves available cities, and scrapes school names, locations, and private candidate acceptance status. The collected data is stored in a structured CSV format for further analysis.

  • ✅ Python
  • ✅ Web Extraction
  • ✅ Data Cleaning

QFC Companies Web Scraper

Built a web scraper using Python, Playwright, and lxml to extract company data from the QFC public register. The script navigates through all pages, collecting firm names (English & Arabic), license status, firm ID, and address. Extracted data is saved into a structured CSV file for further use.

  • ✅ Python
  • ✅ Playwright
  • ✅ TSV/Excel
